Comandi per usare streamlit su Mac



Crea ambiente virtuale
    python -m venv .venv
Attiva ambiente 
    source .venv/bin/activate
Entra nella cartella streamlit-dashboard
    cd ~/Desktop/streamlit-dashboard

Comando unico
        cd ~/Desktop/streamlit-dashboard && source .venv/bin/activate


Attiva streamlit
     Streamline run nomefile.py
